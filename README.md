# Projeto Final IHC 

# Descrição do Projeto

Muitas pessoas perdem o controle financeiro e gastam mais do que deveriam. Para elas poderem ter mais controle de seus gastos e terem mais conhecimento de suas finanças em geral para economizar melhor, essas pessoas podem usar um app que registra receitas e despesas, vê gráficos do quanto gastou por categoria (alimentação, transporte etc.), e recebe alertas quando estiver se aproximando de um limite mensal.

## Funcionalidades principais:
- **Funcionalidade 1**: Cadastro de usuários
- **Funcionalidade 2**: Monitoramento de finanças
- **Funcionalidade 3**: Visualização de despesas
